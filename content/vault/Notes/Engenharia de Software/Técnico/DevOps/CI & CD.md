# Principais Conceitos
O termo DevOps foi criado por Patrick Debois em uma hashtag do Twitter, e a principal ideia é aproximar o desenvolvedor da infraestrutura.

DevOps é uma cultura

CI -> Pratica adotada para executar testes de integração e build da aplicação e enviada para alguma plataforma de armazenamento de artefatos.

Quando o fonte chega no repositório, o CI deve validar se aquele código esta saudável e estável para a release.

Continuous Delivery -> Tudo e criado testado e preparado automaticamente para a liberação em produção
