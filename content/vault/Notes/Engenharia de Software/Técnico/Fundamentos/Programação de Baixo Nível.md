A programação de baixo nível é uma nomenclatura utilizada para programação que nos permite uma manipulação mais macro do sistema, hardware.

As linguagens comuns são:
	C
	C++
	[[Rust Lang]]
	Zig

Essas linguagens nos permite um controle de alguns conceitos importantes, como threads, memoria e afins.
